# LCS(最長共通部分列)

dp[i+1][j+1] := s[i],t[j]まででできるLCSの長さ
・遷移
s[i]==t[j]のとき
dp[i+1][j+1] = dp[i][j] + 1

s[i]!=t[j]のとき
dp[i+1][j+1] = max( dp[i][j+1] , dp[i+1][j])

・初期値
dp[i][j] = 0

・LCSの復元
dp[i][j] - 1 == dp[i-1][j-1] == dp[i-1][j] == dp[i][j-1] のときs[i-1]は使われたということがわかる
このポイントを探す
dp[i][j] == dp[i][j-1] もしくは dp[i][j] == dp[i-1][j] ならその方向に動く(両方ならj-1の方で)
