# 競技プログラミングまとめ

競技プログラミングの解いた問題、ライブラリ置き場

## AtCoder
AtCoderの解いた問題

## Graph
グラフ関係のライブラリ、グラフアルゴリズムを使う問題

## SegTree
セグ木関係のライブラリ、セグ木を使う問題

## Numerical
数学的なアルゴリズムのライブラリ  
- 二項係数
- 高速フーリエ変換 (FFT)
- 高速ゼータ変換 (FZT)
- 行列計算
- 素数やら、最小公倍数とか

## DP
DPを使う問題を種類別に

## Geometrical
幾何アルゴリズムのライブラリ

## Game
Nimとかのゲーム問題

## String
文字列探索系のアルゴリズム
